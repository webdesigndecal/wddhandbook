Introduction

With all our knowledge of CSS, we have still not yet finised the core concepts. A very fundamental part of CSS is its box model, which allows us to add spacing, create beautiful layouts, and shape our meaningless div boxes into something meaningful. Here, we will delve into this box model, talk about the display property, and learn how to use images.



The Box Model

So far, we have only worked with content, but there are some invisible spaces that surround that content. These spaces are called padding, border, and margin. Padding adds space around the inside of a box, within the border. The border is on the edge of the box. Margins add space surrounding the outside of the box.



Image source: https://backlog.com/blog/css-basics-for-engineer-boxmodel/



Let's get into how to use CSS to edit the amount of padding, borders, and margins for each element in your HTML. It's usually a simple addition of the margin, border, or padding property. For example, .div-one { margin-top: 10px; } adds 10 pixels of margin on top of the content box. To add spacing within the border we use padding: .div-one { padding-bottom: 20px; } adds 20 pixels of padding on the bottom. Finally, .div-one { border: 1px solid black } adds a border with a width of 1 pixel to the content box.


Shorthands

There are a number of shorthands for padding, margin, and borders so that you don't have to specify padding-left or margin-bottom or border-right every time you want to add one of those to multiple sides of your content box.

  • .div-one { margin: 10px; } adds 10px of margin to all sides of div-one
  • .div-one { margin: 5px 10px; } adds 5px of margin to top and bottom and 10px to left and right
  • .div-one { margin: 5px 10px 15px 20px; } adds margin in this order: top right bottom left
  • .div-one { padding: 10px; } adds 10px of padding to all sides of div-one
  • .div-one { padding: 5px 10px; } adds 5px of padding to top and bottom and 10px to left and right
  • .div-one { padding: 5px 10px 15px 20px; } adds 10px of padding to all sides of div-one
  • .div-one { border: 1px solid blue; } adds blue border of 1px to all sides
  • .div-one { border: 1px 2px 3px 4px; } adds borders in this order: top right bottom left



Display Property

Now that we understand the box model, we can start thinking about different kinds of displays. The three main ones we'll go over for this week are display: block, display: none, and display: inline-block. display: block makes content stacked on top of each other vertically. display: none removes content from the document and takes them out of the flow of the document as well. In other words, the content will visually disappear and not take up space anymore. display: inline-block puts content next to each other on the same row.


Using Floats

As an alternative to using display: inline-block to position elements next to each other, we can use the float property instead. To float items to the left possible of their container, use float: left. To float items to the farthest right possible of their container, use float: right. In order to stop an element from floating after you've floated everything you've wanted, use clear: left, clear: right, or clear: both. Adding Images

Images can really spice up your website and bring it to life. There are actually two places you can add an image, in the HTML and in the CSS. If the image has a meaning, like a logo or gallery, it would be better to add it to the HTML using the <img src=\"url_to_image\" /> tag. If the image is merely decorative, like a patterned background image, you can use the CSS to add it. The height and width of images can be changed in the CSS, whether it is in the HTML or the CSS, using the height and width CSS properties. To change a background image size, use the background-size property.
